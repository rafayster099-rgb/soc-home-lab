# SOC Home Lab

## Overview
Built a fully functional SOC home lab in
Proxmox to develop hands-on skills in SIEM
deployment, threat detection, and incident
response.

## Lab Architecture
- pfSense (virtual router/firewall)
- Wazuh SIEM (Ubuntu manager + Windows agent)
- Kali Linux (attacker machine)
- Windows Server 2012 (target/monitored endpoint)
